pp#  Hand Gesture Video Controller

Control your media player using hand gestures in real time!  
Built with Python, OpenCV, MediaPipe, and PyAutoGUI.

##  Supported Gestures:
- ✋ Palm → Play/Pause
- ✊ Fist → Esc
- ☝️ Point → Skip Forward
- ✌️ (2 fingers) → Volume Up
- 🤘 (1st and pinky) → Volume Down
- 👍 + ✌️ → Fullscreen

##  How to Run
1. Clone the repo
2. Install dependencies (I already provided a requirments.txt file)
